# Mix Designs

Concrete formulations for canoe hull.

## Format

Each mix design includes:
- Proportions (by weight)
- Water/cement ratio
- Expected density (PCF)
- Expected strength (PSI)
- Cost estimate
- Testing results

## Naming Convention

`MIX-001_descriptive-name.md`

Example: `MIX-001_poraver-heavy-65pcf.md`

## Design Goals

- **Target Density:** 65 PCF
- **Minimum Strength:** 1000 PSI (28 days)
- **Workability:** 4-6 inch slump
- **Cost:** Minimize through donations
