# Tools & Calculators

Python scripts for mix design calculations, cost estimation, etc.

## Planned Scripts

### `mix_calculator.py`
Calculate mix proportions from target density and strength.

```bash
python3 mix_calculator.py --density 65 --strength 1000
```

### `density_predictor.py`
Estimate final density from component specific gravities.

```bash
python3 density_predictor.py --mix MIX-001
```

### `cost_estimator.py`
Calculate material costs and track donations.

```bash
python3 cost_estimator.py --mix MIX-001
```
