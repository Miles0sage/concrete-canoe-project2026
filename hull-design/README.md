# Hull Design

CAD files, mold designs, construction plans.

## Design Requirements

- **Length:** 20 feet (ASCE requirement)
- **Width:** ~30 inches at beam
- **Depth:** ~12 inches
- **Crew:** 2 paddlers
- **Thickness:** 0.25-0.375 inches

## Design Process

1. **Hull Shape Selection** - Choose proven design or modify existing
2. **CAD Modeling** - Create 3D model for analysis
3. **Mold Design** - Plan for negative mold construction
4. **Reinforcement Layout** - Glass fiber mesh placement
5. **Construction Sequence** - Step-by-step fabrication plan

## Files

- `hull-profile.pdf` - Side view, dimensions
- `sections.pdf` - Cross-sections every 2 feet
- `mold-plans.pdf` - Mold construction drawings
- `reinforcement-layout.pdf` - Fiber mesh placement
